# 🛍️ Loja de Roupas

## 📌 Sobre o Projeto

Projeto de e-commerce simples voltado para exibição de produtos de vestuário.

## 🛠️ Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## ⚙️ Funcionalidades

* Listagem de produtos
* Layout responsivo
* Interface amigável

## 🎯 Objetivo

Simular uma loja virtual e praticar conceitos de front-end.

## 📚 Aprendizados

* Estrutura de e-commerce
* Organização visual
* UI básica

## ▶️ Como Executar

```bash
git clone https://github.com/DeivsonRibeiro/Loja_Roupas.git
cd Loja_Roupas
```

Abra no navegador.
